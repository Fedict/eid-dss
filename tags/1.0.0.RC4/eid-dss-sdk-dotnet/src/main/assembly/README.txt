EADME for the FedICT eID Digital Signature Servicei .NET SDK
============================================================

=== 1. Introduction

This package contains the FedICT eID DSS .NET SDK. It includes the following
primary artifacts:

- README.txt                    This file.
- eid-dss-dev-guide.pdf         The eID DSS developer's guide.
- eid-dss-sdk-dotnet            The eID DSS .NET SDK
- DSSTestSP			The eID DSS .NET Test Service Provider WebSite

We suggest to start by reading the eID Digital Signature Service 
developer's guide.
